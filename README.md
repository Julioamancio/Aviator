# Aviator Bot v3.2 Professional

Instalação rápida:
```bash
python installer.py --run
```

Principais Recursos:
- Estratégias múltiplas (threshold, range_spike)
- Editor visual de XPaths
- Gráficos em tempo real (matplotlib)
- Simulador offline (geração ou CSV)
- Perfis de configuração (criar, duplicar, remover)
- Webhooks (Discord + Telegram)
- Licença de 30 dias (renovável via activation_date)
- Bankroll manager (risco %, wins/losses)
- Exportações: CSV / JSON / SQLite
- Screenshots de sinais
- Rotação de logs
- Recuperação automática de WebDriver
- Notificações sonoras / pop-up / webhooks

Renovação de Licença:
Edite license/activation.json e ajuste "activation_date" para a data atual ou implemente validação avançada.

Para atualizar dependências:
```bash
pip install -r requirements.txt --upgrade
```

Uso responsável e educacional.
